Syllabus

HTML

Students will learn how to use HTML to block and build the skeleton of your website. They will start to write a simple website for themselves. By the end of this class, they’ll have a good understanding of HTML.


CSS

Students will learn how to use CSS to improve how your HTML markup looks. You will build upon what your newly built HTML page and turn it into something more presentable.

Javascript

Students will learn Javascript, an essential language for all of web development. Javascript allows you to enhance look and feel or your website. It is a programming language that gives behavior to the elements on your web pages. By the end of the session, you'll have the skills to be a web designer.
